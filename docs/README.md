# MEMORIAE

## Overview
MEMORIAE is a research project submitted to the **ICSRC 2026** conference. It includes a performance evaluation component (mem3) with detailed analysis reports, charts, and architecture documentation.

## Files

| File | Description |
|------|-------------|
| `MEMORIAE_ICSRC2026_Final.docx` | Final conference paper submission |
| `MEMORIAE_ICSRC2026_Final_with_Charts.docx` | Final paper with embedded charts |
| `mem3_final_report.docx` | mem3 evaluation module — final report |
| `mem3_architecture.html` | mem3 system architecture diagram (interactive HTML) |
| `mem3_v2_architecture.png` | mem3 v2 architecture diagram (image) |
| `memdemo1.mp4` | Demo video |
