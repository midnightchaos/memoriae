# Changelog

All notable changes to this project will be documented in this file.

---

## [1.0.0] - 2026-06-13

### Added
- **MIT License**: Added official license text to the repository root.
- **Biometric Security**: Integrated hardware biometric check screen for caregiver authentication.
- **SQLCipher Integration**: Secured sqlite storage with key generation stored in Android KeyStore/iOS Keychain.
- **Menta Chatbot**: Dynamic Gemini-powered conversational assistant with model fallbacks and retry logic.
- **Relaxation Hub**: Breathing guidelines, drawing canvas, guided meditations, and tailored playlist playback.
- **Medication reminders**: Scheduler with local notifications and confirmation/dosage tracking.
- **Daily Agenda**: Checklists for daily routines synced with alarm indicators.
- **System Docs**: Added comprehensive system architecture diagrams (`docs/ARCHITECTURE.md`) and data protection policies (`docs/SECURITY.md`).
- **CI/CD Integration**: Configured automated quality gates via GitHub Actions workflow (`.github/workflows/ci.yml`).
